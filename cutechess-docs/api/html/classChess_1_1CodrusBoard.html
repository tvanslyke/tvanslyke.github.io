Detailed Description

A board for the Game of Codrus, a Losing Chess variant.

The Game of Codrus is a predecessor of Losing Chess.

The King has no royal powers, so there is no check and the King can be captured. Castling is allowed. If a side can capture then they must make a capture move.

The side whose King is captured wins.

The Game of Codrus has been described in J. Brede's Alamanach, Denmark 1844. It is named after the Athenian King Codrus who sacrificed himself to save his people (1068 BC).

Note
Rules: J. Brede (1844): Almanach fr Freunde vom Schachspiel, Altona, (then) Denmark: J. F. Hammreich. p. 82f (in German language). Retrieved from: http://www.binnewirtz.com/Brede.htm.
Also see Pritchard, D. B. (2007). Beasley, John, ed. The Classified Encyclopedia of Chess Variants. Harpenden, UK: John Beasley. ch. 10.9. Retrieved from https://www.jsbeasley.co.uk/encyc.htm.
